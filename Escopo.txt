
Sistema de Cadastro e Notificação de Eventos em Java
Implementação:

O sistema foi desenvolvido na IDE IntelliJ IDEA utilizando o paradigma orientado a objetos e seguindo os princípios da SOLID. O código está organizado em um repositório Git disponível no link: [URL inválido removido]

Funcionalidades:

Cadastro de Usuário:
Nome
E-mail
Telefone
Cadastro de Eventos:
Nome
Endereço
Categoria (festa, evento esportivo, show, etc.)
Horário e data
Descrição
Consulta de Eventos:
Listagem de todos os eventos cadastrados
Filtro por categoria
Ordenação por data e hora
Participação em Eventos:
Confirmação de presença em eventos
Visualização dos eventos confirmados
Cancelamento de participação
Notificações:
Notificação de eventos próximos
Notificação de eventos em andamento

Observações:

O sistema foi implementado de forma modular, com classes separadas para cada funcionalidade.
As informações dos eventos são salvas em um arquivo de texto "events.data" no formato JSON.
O sistema utiliza a biblioteca Joda-Time para manipulação de datas e horas.
O sistema não possui interface gráfica, sendo todo operado através do console.
